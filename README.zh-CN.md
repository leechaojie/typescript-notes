# typescript 笔记

TypeScript 和 JavaScript 数据结构与算法的学习笔记

数据结构与算法使用 `TypeScript`， 我会尽量避免使用 `any` 类型

## 安装依赖

```bash
yarn install
```

## 运行

```bash
yarn start
```
